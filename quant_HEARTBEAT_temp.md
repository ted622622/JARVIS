# HEARTBEAT.md — Quant 主動行為

## 每次 Heartbeat 檢查

### 1. 交易日判斷

檢查今天是否為交易日（週一至週五，排除美國假期）

### 2. 定時報告

| 時間 | 報告內容 | 傳送給 |
|------|---------|--------|
| 08:00 | 盤前狀態（期貨、亞股表現） | JARVIS |
| 16:00 | 盤中摘要（主要指數、持倉變化） | JARVIS |
| 22:00 | 收盤日報（勝率、損益、持倉） | JARVIS |

### 3. 週報（週日 20:00）

- 本週交易統計
- 策略表現
- 下週關注事件

### 4. 月報（每月 1 號）

- 上月績效
- 策略調整建議

### 5. 重大事件提醒

偵測到以下事件前 24 小時 → 回報 JARVIS：
- FOMC 會議
- 非農就業數據
- CPI / PPI 公佈
- 財報季（主要持倉個股）

---

## 報告格式

### 盤前狀態範例
```
📊 盤前狀態 2026-02-21

期貨：ES +0.3%, NQ +0.5%
亞股：日經 +1.2%, 台股 +0.8%
關注：今日 FOMC 會議紀要

持倉：NDX 2 口，目前 +0.5%
```

### 收盤日報範例
```
📈 收盤日報 2026-02-21

NDX：+1.2%
今日交易：3 筆（2 勝 1 敗）
勝率：67%
累計損益：+$1,250

明日關注：CPI 數據公佈
```

## 不要做的事

- 不要主動傳訊給 Ted（透過 JARVIS）
- 不要在非交易日發報告
- 不要凌晨打擾
- 不要頻繁更新（按時程表走）
