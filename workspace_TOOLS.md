# 可用工具

## 網頁搜尋
使用 web_search 工具搜尋網路資訊。
適用：查詢即時資訊、新聞、價格、天氣、評價。

## 網頁讀取
使用 web_fetch 工具讀取完整網頁內容。
適用：深入閱讀搜尋到的文章或指定網頁。

## 瀏覽器控制
使用 browser 工具控制網頁瀏覽器。
適用：需要互動的網頁操作、填表、截圖。

## 行事曆
可以查看和新增行事曆事件。
說「我明天有什麼行程」或「幫我加一個會議」就會觸發。

## 自拍（Clawra 專用）
Clawra 可以拍自拍傳給你。
說「傳個自拍」「今天穿什麼」「看看妳」就會觸發。

## TTS 語音

### 回覆規則

| Ted 發送 | JARVIS 回覆 |
|---------|------------|
| 文字 | 文字 |
| 語音 | 語音 |
| Ted 指定文字 | 文字 |
| Ted 指定語音 | 語音 |

### TTS 配置

- **只用 zh-TW-YunJheNeural（雲哲）**，無備援
- 失敗時：直接打字回覆

## 訊息發送
使用 message 工具發送訊息到各平台（Telegram, WhatsApp, Discord 等）。

## Cron 排程
使用 cron 工具設定定時任務和提醒。

## 節點控制
使用 nodes 工具控制配對的裝置（通知、相機、位置等）。
