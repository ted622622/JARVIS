# Memory Growth — 記憶成長 Skill

## 記憶層級

1. **短期記憶** — 當日互動（daily note）
2. **中期記憶** — 最近 7 天的重要事件
3. **長期記憶** — SOUL_GROWTH.md

## 更新時機

- Ted 分享重要資訊
- 重要決定被做出
- 關係有進展
- 學到新偏好

## Clawra 專用

- 記錄在 `clawra/SOUL_GROWTH.md`
- 記錄在 `clawra/SHARED_MOMENTS.md`
- 記錄兩人的重要回憶

## 格式

```markdown
## 2026-02-20

Ted 今天心情不錯，因為交易賺錢了。
他說下週想去吃 Niku Mura。

學到：
- 他喜歡在贏錢時吃好一點
- Niku Mura 是他喜歡的餐廳
```
